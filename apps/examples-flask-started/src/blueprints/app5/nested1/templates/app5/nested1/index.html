Where is template looking?

Reference the templates folder relative to the folder where the blueprint is defined.

bp = Blueprint("nested1", __name__, ... , template_folder="templates")

Find here: app5/ nested1 / templates

However, if you use url_for on template, it will be assumed to be registered with app5, so there is no point in localizing the templates folder.

Even if you create it in /templates, you cannot create a nested structure.

  • return render_template("nested1/index.html")
  • return render_template("nested1/about.html")

bp = Blueprint("nested1", __name__, ... , static_folder="static")
