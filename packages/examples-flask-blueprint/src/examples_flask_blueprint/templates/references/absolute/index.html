Where does the template folder reference?

If you specify it from the root, only the /templates will be referenced.

bp = Blueprint("reference_absolute", __name__, template_folder="/templates")

render_template() references /templates.

  • return render_template("references/absolute/index.html")
  • return render_template("references/absolute/about.html") (TemplateNotFound)

Where is static looking?

If you specify it from the root, only the /static will be referenced.

bp = Blueprint("reference_absolute", __name__, static_folder="/static")

This specification does not result in an error, but it seems that it cannot be referenced.

  • url_for('reference_absolute.static', filename='...')

This code are valid:

  • url_for('static', filename='...')
is applied.
