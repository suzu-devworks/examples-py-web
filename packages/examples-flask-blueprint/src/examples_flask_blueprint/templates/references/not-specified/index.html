Where does the template folder reference?

If not specified, only the /templates will be referenced.

bp = Blueprint("reference_not_specified", __name_)

render_template() references /templates.

  • return render_template("references/not-specified/index.html")
  • return render_template("references/not-specified/about.html") (TemplateNotFound)

Where does the static folder reference?

If not specified, only the /static will be referenced.

bp = Blueprint("reference_not_specified", __name__)

The following cannot be specified because the endpoint will not be generated:

  • url_for('reference_not_specified.static', filename='...')

This code are valid:

  • url_for('static', filename='...')
is applied.
