Where does the template folder reference?

It references a folder relative to the folder where the blueprint is defined, but can be overridden by any folder starting from the root.

bp = Blueprint("reference_relative", __name__, template_folder="templates")

render_template() references reference_relative/templates.

However, if the same name exists in the /templates folder, that seems to take precedence.

  • return render_template("references/relative/index.html")
  • return render_template("references/relative/about.html")

Therefore, the official recommendation is to specify the folder using the blueprint name.

Where does the static folder reference?

In the case of static folders, overwriting is not possible.

bp = Blueprint("reference_relative", __name__, static_folder="static")

These codes are valid:

  • url_for('static', filename='...')
  • url_for('reference_relative.static', filename='...')
